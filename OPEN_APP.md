# Open Swift Coach locally

**[Click here to open http://localhost:3000 in Cursor's browser](command:simpleBrowser.show?%5B%22http%3A%2F%2Flocalhost%3A3000%22%5D)**

If that does not work: `Ctrl+Shift+P` → **Browser: Open Integrated Browser** → enter `http://localhost:3000`

Or run task: `Ctrl+Shift+P` → **Tasks: Run Task** → **Open Swift Coach (localhost)**
